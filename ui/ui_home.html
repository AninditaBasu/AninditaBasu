UI portfolio
User interface, user interaction, wireframes

ReportGen
Wireframes for a report generation software. This enterprise-level software is for creating reports that contain data pulled from disparate sources. These data sources can either be documents themselves (in .doc, .pdf., or .xml formats) or be other documents created through this software. Parts (or sections) of a report can contain data from more than one source.

Kindle
Wireframes to show how I'd like my Kindle Reader to be. The user tasks focussed on are: browsing a library; referring to bookmarks, notes, and highlights; adding books to collections; and exploring a book, annotating, and sharing while reading it.
